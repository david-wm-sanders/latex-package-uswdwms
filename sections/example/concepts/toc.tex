\subsection{Tables of Contents}
Tables of contents are remarkably easy to create in \LaTeX\ -- with sections declared as described in Section~\ref{primer:sec:concepts:sectioning}, it is possible to use the \term{\textbackslash tableofcontents} command \textit{(as shown in Listing~\ref{primer:lst:tableofcontents})} to create a table of contents that will be populated automatically as a document is rendered.

With the \texttt{hyperref} package, included and configured by default by \uswdwmspkg{}, table of contents entries will be automatically hyperlinked.

\begin{listing}[H]
  \captionsetup{skip=\skiplistingcaptionlen}
  \begin{minted}[highlightlines={3}]{tex}
    \begin{document}
    \input{sections/_usw_titlepage.tex}
    \tableofcontents
  \end{minted}
  \caption{\texttt{\textbackslash tableofcontents} command example}
  \label{primer:lst:tableofcontents}
\end{listing}
