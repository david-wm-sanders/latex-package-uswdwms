\subsection{Tables of Contents}
Tables of contents are remarkably easy to create in \LaTeX\ -- with section levels as described in Section~\ref{example:sect:concepts:sectioning}, it is possible to dynamically create a table of contents with the \term{\textbackslash tableofcontents} command \textit{(as shown in Listing~\ref{example:lst:tableofcontents})}. The table of contents will be automatically updated as new sections are added to a document.

With the \texttt{hyperref} package, included and configured by default by \uswdwmspkg{}, table of contents entries will be automatically hyperlinked to the page for the section.

\begin{listing}[H]
  \captionsetup{skip=\skiplistingcaptionlen}
  \begin{minted}{tex}
    \begin{document}
    \input{sections/_usw_titlepage.tex}
    \tableofcontents
  \end{minted}
  \caption{\texttt{\textbackslash tableofcontents} command example}
  \label{example:lst:tableofcontents}
\end{listing}
